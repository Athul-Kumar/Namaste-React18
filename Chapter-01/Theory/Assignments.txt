2.Difference between a Library and Framework?

Library and Framework both are created to make software development lifecycle easy!

a.Library --> provides collection of functions or code snippets to use on our need without
              worring about flow of the program or architecture of the App.

b.Framework --> provides collection of functions or code snippets to use on our need but we have
                to follow certain rules that won't affect the flow of the program or architecture
                of the App.
